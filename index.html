2. 基礎單字與問候語 (Word Foundation)

學習最基本的問候語和核心詞彙！

常用問候語

  • **안녕하세요** (An-nyeong-ha-se-yo) - 您好 / 請問好 (最常用敬語)
  • **감사합니다** (Gam-sa-ham-ni-da) - 謝謝 (敬語)
  • **네 / 아니요** (Ne / A-ni-yo) - 是 / 不是
  • **죄송합니다** (Joe-song-ham-ni-da) - 對不起

核心基礎詞彙

  • **나라** (Na-ra) - 國家
  • **사랑** (Sa-rang) - 愛
  • **친구** (Chin-gu) - 朋友
  • **가족** (Ga-jok) - 家人
